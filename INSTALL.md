# 설치 방법

> 크롬 웹 스토어에 등록된 버전이 아니라, 직접 파일을 로드하는 방식입니다.
> GitHub를 처음 사용하더라도 아래 순서대로 따라하면 됩니다.

## 1단계 — 파일 다운로드

1. [Releases 페이지](https://github.com/mintheon/playlist-replug/releases/latest)로 이동

2. 최신 버전 **Assets** 항목에서 `playlist-replug-vX.X.X.zip` 다운로드

3. ZIP 파일 압축 해제

## 2단계 — 크롬에 로드

1. 크롬 주소창에 입력 후 엔터

   ```
   chrome://extensions
   ```

2. 우측 상단 **개발자 모드** 토글 켜기

3. **압축해제된 확장 프로그램을 로드합니다** 클릭

4. 압축 해제한 폴더 선택

5. 크롬 툴바에 Playlist Replug 아이콘이 나타나면 완료

## 3단계 — 사용 전 준비

- 유튜브에 로그인된 탭이 열려있어야 합니다
- 아이콘이 안 보이면 툴바 우측 퍼즐 아이콘 → Playlist Replug 옆 핀 클릭

---

## 업데이트

1. [Releases 페이지](https://github.com/mintheon/playlist-replug/releases/latest)에서 최신 ZIP 다운로드 후 압축 해제
2. 기존 폴더를 새 파일로 덮어쓰기
3. `chrome://extensions`에서 Playlist Replug 카드의 새로고침(↺) 클릭

---

## 자주 묻는 질문

**"압축해제된 확장 프로그램을 로드합니다" 버튼이 안 보여요**  
→ 개발자 모드 토글을 먼저 켜야 버튼이 나타납니다.

**설치 후 아이콘이 안 보여요**  
→ 툴바 우측 끝 퍼즐 아이콘 클릭 → Playlist Replug 찾아서 핀 클릭

**아이콘을 클릭해도 창이 안 떠요**  
→ 일부 로딩 시간이 있습니다. 몇 초 기다렸다가 다시 클릭하세요.

**변환 속도가 느려요**  
→ 유튜브 동영상 검색 및 좋은 품질의 동영상 추가를 위한 우선순위 반영과 검색 매칭이 필요하기 때문입니다. 곡당 약 1~2초가 소요됩니다.
